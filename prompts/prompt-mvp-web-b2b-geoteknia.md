# OBJETIVO
Tu objetivo es definir las funcionalidades CORE (MVP) para una nueva web corporativa B2B de captación de clientes de una empresa de ingeniería de servicios geotécnicos en España. El objetivo principal de la web es la generación de leads cualificados (peticiones de presupuesto, contactos técnicos y licitaciones) a través de tráfico orgánico (SEO) y de pago.
Las empresas de este sector ofrecen servicios altamente técnicos como estudios geotécnicos para edificación, sondeos mecánicos, ensayos de laboratorio, informes periciales y geotecnia portuaria/marítima. La audiencia son arquitectos, promotoras, ingenierías civiles y administraciones públicas que buscan solvencia técnica, cobertura geográfica, experiencia demostrable y facilidad de contacto.
